- 언리얼의 게임 오브젝트를 처리하기 위한 클래스.
- 언리얼 오브젝트에 대한 베이스 클래스는 __UObject__임
- __UCLASS__ 매크로는 __UObject__에서 파생된 클래스에 태그를 지정해, UObject처리 시스템이 인식하도록 할 수 있음.

## UCLASS 매크로
- 언리얼 기반 타입을 설명하는 UCLASS에 대한 레퍼런스를 UObject에 제공.
- 각 UCLASS는 CDO(Class Default Object)라는 하나의 오브젝트를 유지함.
- CDO는 기본적으로 클래스 생성자에 의해 생성되고, 이후에는 수정되지 않는 Default 템플릿 오브젝트.  
<br>

- UCLASS와 CDO는 보통 읽기 전용으로 간주돼야 하나, 주어진 오브젝트 인스턴스에 대해 얻을 수 있음. 
- 오브젝트 인스턴스에 대한 UCLASS는 GetClass()함수를 사용해 엑세스 가능.  
<br> 
- UCLASS에는 클래스를 정의하는 프로퍼티와 함수세트가 포함됨.
- 이는 표준 c++코드에 사용가능한 일반 c++ 함수 및 변수이나, 오브젝트 시스템 내에서 행동하는 방식을 제어하는 언리얼 엔진 전용 메타데이터로 태그가 지정되어있음.
